# 🚀 Proyectos

## 1️⃣ IntegraDav – Plataforma de Soluciones Digitales

Plataforma web institucional diseñada para presentar servicios tecnológicos orientados a empresas y profesionales que buscan mejorar su presencia digital y optimizar procesos mediante herramientas tecnológicas.

La plataforma funciona como una **vitrina tecnológica** para comunicar soluciones digitales como desarrollo web, automatización de procesos y sistemas personalizados.

### Tecnologías utilizadas

- React
- Next.js
- JavaScript
- HTML / CSS
- SEO on-page
- Integraciones web

### Funcionalidades

- Página institucional de presentación de la iniciativa
- Sección de servicios tecnológicos
- Sección de proyectos desarrollados
- Integración con canales de contacto
- Optimización para motores de búsqueda
- Estructura orientada a captación de clientes

### Resultado

Se desarrolló una plataforma que permite presentar servicios tecnológicos de forma profesional y facilitar el contacto con potenciales clientes interesados en soluciones digitales.

## 2️⃣ Desarrollo Web y SEO – Justicia y Valor

Proyecto de desarrollo web profesional para el despacho jurídico **Justicia y Valor**, especializado en derecho penal, disciplinario y administrativo en Medellín.

El objetivo fue mejorar la presencia digital del despacho y facilitar la captación de clientes mediante posicionamiento en buscadores.

### Tecnologías utilizadas

- Next.js
- React
- SEO on-page
- Google Maps
- Diseño responsive

### Funcionalidades

- Desarrollo completo del sitio web profesional
- Estructuración de contenido optimizado para SEO
- Secciones de servicios jurídicos
- Integración con Google Maps
- Sistema de contacto directo
- Optimización de experiencia de usuario

### Resultado

El sitio web logró indexarse correctamente en Google y aparecer en resultados de búsqueda relacionados con servicios legales en Medellín, fortaleciendo la presencia digital del despacho.
